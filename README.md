# JavaReport

とりあえず載せてみました  

### javaを作る上での注意点
- ファイル名と実行クラスは同じ名前に
- ファイル名およびクラス名は最初大文字

### VScodeでjavaを使う場合
  [JDK](https://www.oracle.com/java/technologies/javase-jdk15-downloads.html)をインストールし実行した後PATHを通してください。<br>
  デフォルトで
`C:\Program Files\Java\jdk-15.0.1\bin`
